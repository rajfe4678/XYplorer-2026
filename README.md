## 🔧 Common Tasks

### Copying Files Between Folders

1. Navigate to your source folder in the left pane
2. Navigate to your destination folder in the right pane
3. Select the files you want to copy
4. Press Ctrl+C to copy, then Ctrl+V to paste in the destination
5. Alternatively, drag files from one pane to the other

### Searching for Files

1. Press Alt+F or click the search icon
2. Type your search term in the search box
3. Use filters to narrow results (by date, size, type)
4. Double-click any result to open it

### Renaming Multiple Files

1. Select the files you want to rename
2. Press Ctrl+M or right-click and choose "Rename"
3. Enter your naming pattern using tags like {n} for numbers
4. Preview the changes before applying
5. Click "Rename" to complete the operation

### Using Tabs

XYplorer tabs work like browser tabs:

- Click the "+" button to open a new tab
- Ctrl+T creates a new tab in the active pane
- Ctrl+W closes the current tab
- Drag tabs to reorder them
- Right-click a tab for additional options

## 🛠️ Troubleshooting

**The program does not start after installation:**
